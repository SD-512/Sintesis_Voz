# Aplicación de Síntesis de Voz

## Descripción

Este proyecto fue desarrollado en Python utilizando la biblioteca `pyttsx3`.

La aplicación permite que el usuario escriba un mensaje y luego lo reproduce en voz alta mediante la síntesis de voz del sistema operativo.

## Tecnologías utilizadas

- Python 3
- pyttsx3

## Instalación

1. Clonar el repositorio.
2. Instalar la biblioteca:

```bash
pip install pyttsx3
```

3. Ejecutar el programa:

```bash
python main.py
```

## Uso

Al iniciar el programa, el usuario escribe un texto en la consola y la aplicación lo lee en voz alta.
